#  House Price Prediction

## Overview
End-to-end Machine Learning project to predict house prices
using Linear Regression.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Steps
- Data Loading & EDA
- Data Cleaning & Outlier Removal
- Feature Engineering
- Model Building (Linear Regression)
- Model Evaluation (MAE, RMSE, R2)
## Results
| Metric | Score |
|--------|-------|
| R2 Score | 0.68 |
| MAE | $1,970 |
| RMSE | $14,100 |


## Dataset
California Housing Dataset from Kaggle
